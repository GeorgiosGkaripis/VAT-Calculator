## 📝 Περιγραφή
Αυτό το εργαλείο αναπτύχθηκε για τον αποτελεσματικό υπολογισμό και τη σύγκριση οικονομικών δεδομένων σε τεχνικά έργα. Επιτρέπει στον χρήστη να αναλύει τη σχέση μεταξύ του **Financial Audit** (Οικονομικός Έλεγχος) και του **Value Efficiency** (Αποδοτικότητα Αξίας), διευκολύνοντας τη λήψη αποφάσεων κατά τη διάρκεια της κατασκευής.

## 🚀 Κύριες Λειτουργίες
* **Υπολογισμός ΦΠΑ (VAT Calculator):** Αυτόματη εξαγωγή καθαρής αξίας και φόρου από τελικές τιμές ή το αντίστροφο.
* **Ανάλυση Αποδοτικότητας:** Σύγκριση κόστους-οφέλους σε πραγματικό χρόνο.
* **Διασφάλιση Δεδομένων:** Ο κώδικας έχει υποστεί βελτιστοποίηση και θωράκιση (obfuscation) για την προστασία των υπολογιστικών μοντέλων.
